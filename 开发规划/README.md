# 开发规划总览

本目录按**开发顺序**拆分阶段说明：每一阶段的**开发目标**、**交付物**与**验收要点**。请按编号从小到大依次推进；后一阶段依赖前一阶段的约定（尤其是 **chunk 统一 ID**、**切分策略**与基线检索配置）。若调整约定，**直接修改对应阶段 md** 即可。

## 技术栈

**摘要**：Chunk 切分 **Ollama** `qwen2:1.5b`；Embedding **`mxbai-embed-large:latest`**；主 LLM **`glm-5`**（OpenAI 兼容 API）；重排 **`jina-reranker-v2`**。全栈表、接入说明与剩余待定项见 [技术栈约定.md](./技术栈约定.md)。各阶段另有 **「本阶段技术栈」** 小节。

**Python RAG 代码位置**：仓库子目录 **`juyao-agentic-rag/`**，依赖清单为同目录 **`requirements.txt`**（不使用 `pyproject.toml`，安装：`pip install -e .`）。
当前运行配置采用硬编码模式：直接在 `juyao-agentic-rag/src/rag_core/config.py` 修改模型与服务参数。

| `未开始` | 尚未动工 |
